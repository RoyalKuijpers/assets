# Styles

This directory contains style-related assets like CSS variables, color palettes, and design tokens.

## Structure

- `colors.css` - Color palette definitions
- `variables.css` - CSS custom properties
- `tokens.json` - Design tokens in JSON format

## Usage

Import styles in your project:

```javascript
import '@royalkuijpers/kuijpers-assets/assets/styles/colors.css';
```
